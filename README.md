# 📊 Data Analyst Roadmap

A comprehensive, interactive roadmap for aspiring data analysts - starting with the most important tool in the industry: **Excel & Spreadsheets**.

## 📖 Overview

This repository hosts an interactive learning website designed to guide beginners through the essential skills needed for a career in data analysis. The roadmap starts with Excel/Google Sheets, which remain the primary tools used by analysts in most companies, especially in India.

**Live Site:** [https://adminrajeev007.github.io/da-roadmap/](https://adminrajeev007.github.io/da-roadmap/)

## 🎯 Current Module: Excel & Spreadsheets

### Why Excel First?
> "The analyst's first weapon — master before anything else"

- **Industry Reality**: Even in 2025, Excel/Google Sheets is the #1 tool used by analysts
- **Practical Application**: 80% of internship tasks involve cleaning, summarizing and presenting data in Excel
- **Universal Format**: Your manager will share a `.xlsx` file before a Python notebook
